# 赛博粒子交互

在线互动特效工具站，提供告白祝福、节日卡片、赛博小游戏、手势粒子、魔法阵和剑阵视觉模板。

Public site:

https://huyansheng3.github.io/cyber-particle-interaction/

Sitemap:

https://huyansheng3.github.io/cyber-particle-interaction/sitemap.xml

## Features

- Interactive blessing and confession pages.
- Custom share links with recipient name and message.
- Share tools for link copy, system share, WeChat, Xiaohongshu, Weibo, and share-card download.
- SEO metadata, canonical URLs, Open Graph image, `robots.txt`, and `sitemap.xml`.
- Trust pages: about, privacy policy, terms, and contact.
- Ad placement placeholders away from primary game and gesture controls.

## Main Pages

- Homepage: `index.html`
- 520 confession particles: `love-520.html`
- Silk heart confession: `confession-heart.html`
- Holiday wish tree: `holiday-tree.html`
- Cyber snake game: `cyber-snake.html`
- 3D gesture particles: `particle-dream.html`
- Magic circle effects: `magic/`
